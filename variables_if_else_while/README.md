
# C - Variables, if, else, while

This project covers the fundamentals of C programming including variables, conditional statements, and loops.

## Learning Objectives

- Understanding arithmetic operators in C
- Working with logical and relational operators
- Using if, if...else statements
- Implementing while loops
- Declaring and using variables
- Printing variables using printf
- Understanding ASCII character set
- Working with different data types (char, int)

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.c` | Program that determines if a random number is positive, negative, or zero |
| `1-last_digit.c` | Program that prints the last digit of a random number |
| `2-print_alphabet.c` | Program that prints the alphabet in lowercase |
| `3-print_alphabets.c` | Program that prints the alphabet in lowercase then uppercase |
| `4-print_alphabt.c` | Program that prints the alphabet except q and e |
| `5-print_numbers.c` | Program that prints single digit numbers from 0 to 9 |
| `6-print_numberz.c` | Program that prints single digit numbers using putchar |
| `7-print_tebahpla.c` | Program that prints the alphabet in reverse |
| `8-print_base16.c` | Program that prints all numbers of base 16 in lowercase |
| `9-print_comb.c` | Program that prints all possible combinations of single-digit numbers |

## Compilation

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 [filename].c -o [output_name]
```

## Usage Examples

```bash
# Compile and run positive/negative checker
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 0-positive_or_negative.c -o positive_or_negative
./positive_or_negative

# Compile and run alphabet printer
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 2-print_alphabet.c -o print_alphabet
./print_alphabet
```
