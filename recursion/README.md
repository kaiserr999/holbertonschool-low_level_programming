# Recursion

## Task 0: She locked away a secret, deep inside herself, something she once knew to be true... but chose to forget

Write a function that prints a string, followed by a new line, using recursion.

### Prototype
```c
void _puts_recursion(char *s);
```

### Requirements
- Uses recursion to print each character
- Prints a newline at the end
- Handles NULL pointer gracefully

### Files
- `main.h` - Header file with function declarations
- `_putchar.c` - Character output function (writes to stdout)
- `0-puts_recursion.c` - Recursive string printing implementation
